# Description

Basic example for deploying a license plate detection model using the Streamlit tool.

This is designed for the participants of Talento Tech's 2026 Innovative AI Bootcamp.

And Silver for any model obtained with YOLO.

# Steps

**Test locally**

1. Clone this repository
2. Create and activate the virtual environment using the requirements.txt document
3. Use the command streamlit run app.py
4. Open url http://localhost:8501

**Deploy in Streamlit**

1. Upload your project to your own repository
2. Create a Streamlit account with GitHub (preferably)
3. Press the "Create app" button from Streamlit
4. Choose the option to deploy with GitHub
5. Fill out the form with the project information: URL, branch name, main file name
6. Go to advanced settings and select Python 3.10
7. Save settings and deploy
